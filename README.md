## 🎯 Engineering Mindset

- ✅ **Correctness first** - Write code that works reliably
- 📚 **Treat bugs as learning signals** - Every bug is an opportunity to improve
- 📝 **Write self-explanatory code** - Code should communicate intent without comments
- 🔧 **Maintainable over clever** - Prioritize long-term sustainability
